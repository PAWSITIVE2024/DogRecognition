# 🐶강아지 얼굴 인식 및 구별

**강아지 얼굴에서 특징점을 찾고 이를 기억해서 누가 누구인지 판단하는 알고리즘**

원본 사진으로 진행시 GPU 8GB이상 필요

사진을 resize해서 GPU 0.5GB까지 줄이기 성공

known_face에 넣을 사진은 되도록이면 표정이 없는 정면 사진. 옆면은 인지불가.
